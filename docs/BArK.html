Temporary instructions:

Please remember that BArK is still at the very early test stages. You MUST keep a safe copy of your game before you use this tool.
  • Select Games in the explorer
  • Select Import Game in the toolbox
  • Paste your gamedata, or game html into the textbox or drag a file into the textbox
  • Your game should appear in the explorer, select your tile, sprite or item
  • Select Edit Animation in the toolbox
  • Edit your animation
  • Select your game in the explorer
  • Select Game Data in the toolbox
  • You can cut and paste your game data or your html file (If you started with a whole html file) from the tool window
